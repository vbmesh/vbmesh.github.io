Welcome to VBMesh.net

Want to show up on the Mesh Map? Follow these steps to set up your node:

Channels:

Select "Uplink enabled" in your primary channel to share info about other nodes seen, given that they have "okay to mqtt" enabled.

Device:

- Role: select "client" or "client mute"

Lora Settings:

- Number of Hops: set this to 3 or 4. 4-5 hops can be okay if you are using a client_mute role. - "OK to MQTT" turned on: this allows the packets sent from your node to be relayed to other nodes with mqtt, and if you want to show up on the map.

MQTT Settings:

- Address: mqtt.vbmesh.net

- Port: 1883

- User: meshdev

- Password: large4cats

- Encryption enabled: yes

- Root topic: msh/US/VBmesh

- Enable Proxy to Client to allow the node to get internet through your client
