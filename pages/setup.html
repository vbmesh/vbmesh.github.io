Welcome to VBMesh.net

Want to show up on the Mesh Map? Follow these steps to set up your node:

Lora Settings: - "OK to MQTT" turned on: this allows the packets sent from your node to be relayed to the vbmesh server

MQT Settings: - Address: mqtt.vbmesh.net - Port: 1883 - User: meshdev - Password: large4cats - Encryption enabled: yes - Root topic: msh/US/VBmesh - Enable Proxy to Client to allow the node to get internet through your client
