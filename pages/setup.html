Welcome to VBMesh.net

Want to show up on the Mesh Map? Follow these steps to set up your node:

Channels:

Select "Uplink enabled" in your primary channel (for most, this is probably LongFast)to share info about other nodes seen, given that they have "okay to mqtt" enabled.

Device:

- Role: select "client" or "client_mute"

Location:

- Tip: if your node doesn't have a GPS module or is fixed (not mobile or handheld), turn on "fixed position", and enter the node's latitude and longitude.

Lora Settings:

- Modem Preset: LongFast

- Region: US915

- Hop Limit: set this to 3 or 4. 4-5 hops can be okay if you are using a client_mute role. - "OK to MQTT" turned on: this allows the packets sent from your node to be relayed to other nodes with mqtt, and if you want to show up on the map.

MQTT Settings: (follow these instructions to set up your node as a source for vbmesh data

- Address: mqtt.vbmesh.net

- Port: 1883

- User: meshdev

- Password: large4cats

- Encryption enabled: yes

- Root topic: msh/US/VA/VBmesh

- Enable Proxy to Client to allow the node to get internet through your client
